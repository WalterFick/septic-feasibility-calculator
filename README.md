# Septic Feasibility Web App

A simple React + Vite web app for internal septic feasibility screening.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm install
npm run build
```

The production files will be created in `dist/`.

## Deploy to Vercel

1. Create a new project in Vercel.
2. Upload this folder or connect the repository.
3. Framework preset: `Vite`
4. Build command: `npm run build`
5. Output directory: `dist`

## Deploy to Netlify

1. Create a new site in Netlify.
2. Upload this folder or connect the repository.
3. Build command: `npm run build`
4. Publish directory: `dist`

## Notes

This is an internal screening tool only. It does not replace septic engineering review or Department of Health approval.
